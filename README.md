# 📊 Real Estate Marketing Analytics Dashboard

This repository contains a professional **Power BI Dashboard** designed to track and analyze Facebook Ads performance for **Lmar Marketing Real Estate Company**.

## 🚀 Project Overview
The goal of this project is to transform raw Meta Ads data into actionable insights. By monitoring key metrics like Reach, Spend, and Messaging Results, the company can optimize its marketing budget and increase ROI.

## 🛠️ Tools & Technologies
- **Power BI Desktop:** Used for data visualization and DAX measures.
- **Microsoft Excel:** Served as the primary data source.
- **GitHub:** For version control and portfolio hosting.
